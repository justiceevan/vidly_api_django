# Vidly Api Django

This is a simple movie api. It is a clone of the vidly-api-node

- The api has the following endpoints:

| Endpoint              | Method |
| --------------------- | ------ |
| api/genres            | GET    |
| api/movies            | GET    |
| api/movies/id         | GET    |
| api/movies/create/    | POST   |
| api/movies/id/update/ | PUT    |
| api/movies/id/delete/ | DELETE |
